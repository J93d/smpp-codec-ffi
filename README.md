# smpp-codec-ffi

FFI bindings for the `smpp-codec` crate, powered by [UniFFI](https://github.com/mozilla/uniffi-rs).

This crate provides a mechanism to use the `smpp-codec` library from other languages (like Python, Swift, Kotlin, etc.) by defining a UniFFI interface.

> [!WARNING]
> **Disclaimer**:Portions of this codebase were generated by Google's Gemini AI. 
> While efforts have been made to ensure correctness, users should review and test the code thoroughly before using it in production environments.

## Usage

This crate is intended to be used to generate bindings.

To generate bindings for a specific language (e.g., Python), you can use the `uniffi-bindgen` CLI tool or integrate it into your build process.
