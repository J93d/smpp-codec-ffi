# Kotlin Examples

This directory contains examples for using the `smpp-codec-ffi` Kotlin bindings.

## Prerequisites

1.  **Generated Bindings**: You need the `uniffi/smpp_codec_ffi` folder generated by the CI (or locally).
2.  **Shared Library**: You need the compiled shared library (`libsmpp_codec_ffi.so` / `.dylib` / `.dll`).

## Compilation and Running

To compile and run this example, you need `kotlinc` and `java` installed.

1.  Copy the `uniffi` folder from `bindings/kotlin/` to this directory.
2.  Copy the shared library to this directory.
3.  Compile:
    ```bash
    kotlinc uniffi/smpp_codec_ffi/smpp_codec_ffi.kt TestUsage.kt -include-runtime -d test_usage.jar
    ```
### Tool Client/Server

1.  **Compile**:
    ```bash
    kotlinc uniffi/smpp_codec_ffi/smpp_codec_ffi.kt ToolServer.kt -include-runtime -d tool_server.jar
    kotlinc uniffi/smpp_codec_ffi/smpp_codec_ffi.kt ToolClient.kt -include-runtime -d tool_client.jar
    ```

2.  **Run Server**:
    ```bash
    java -jar tool_server.jar
    ```

3.  **Run Client**:
    ```bash
    java -jar tool_client.jar
    ```
